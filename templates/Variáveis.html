Tópico:Variáveis em C#

As variáveis são componentes fundamentais em programação, usadas para armazenar e manipular dados. Elas têm nome, valor e tipo associados, e são utilizadas para representar informações diversas dentro do programa.

Subtópico: Nomes

Os nomes das variáveis devem seguir regras específicas, incluindo tamanho e forma permitidos. Algumas palavras são reservadas e não podem ser usadas como nomes de variáveis.

Subtópico: Variáveis

As variáveis têm três componentes principais: endereço (localização na memória), valor (dados armazenados) e tipo (natureza dos dados).

Subtópico: Amarração

A amarração se refere à associação entre o nome da variável, seu valor e tipo. Pode ser estática (fixada durante a compilação) ou dinâmica (fixada em tempo de execução).

Subtópico: Inferência de Tipo

A inferência de tipo permite que o compilador determine automaticamente o tipo da variável com base no valor atribuído a ela.

Subtópico: Amarrações de Armazenagem

As amarrações de armazenamento se referem à forma como as variáveis são armazenadas na memória. Isso inclui armazenamento estático (em tempo de compilação), na pilha (para variáveis locais) e no heap (para objetos dinamicamente alocados).

Subtópico: Escopo

O escopo determina onde uma variável pode ser acessada e manipulada. Pode ser estático (definido em subprogramas, blocos ou globalmente) ou dinâmico (determinado pelo fluxo de execução).

Subtópico: Ambiente de Referenciação

O ambiente de referênciação é o conjunto de todas as variáveis acessíveis em um determinado ponto do programa.

Subtópico: Constantes

As constantes são valores fixos que não podem ser alterados durante a execução do programa. Elas fornecem uma maneira de nomear valores importantes e reduzir a possibilidade de erros.

EXEMPLOS DE CODIGO

Exemplo de Código: Variáveis e Constantes em C#

using System;

class Program {
static void Main(string[] args) {
// Variáveis
int idade = 25;
double altura = 1.75;
string nome = "João";

Console.WriteLine("Nome: " + nome);
Console.WriteLine("Idade: " + idade);
Console.WriteLine("Altura: " + altura);

// Constantes
const double PI = 3.14159;
Console.WriteLine("Valor de PI: " + PI);

// Inferência de Tipo
var salario = 1500.50;
Console.WriteLine("Salário: " + salario);
}
}

Exemplo de Código: Escopo em C#

using System;

class Program {
static int variavelGlobal = 10;

static void Metodo() {
int variavelLocal = 5;
Console.WriteLine("Variável Local: " + variavelLocal);
Console.WriteLine("Variável Global: " + variavelGlobal);
}

static void Main(string[] args) {
Metodo();
// Console.WriteLine("Variável Local: " + variavelLocal); // Erro: variávelLocal não é acessível aqui
Console.WriteLine("Variável Global: " + variavelGlobal);
}
}

Exemplo de Código: Amarração de Armazenamento em C#

using System;

class Program {
static int variavelGlobal = 10;

static void Metodo() {
int variavelLocal = 5;
Console.WriteLine("Variável Local: " + variavelLocal);
}

static void Main(string[] args) {
Metodo();
Console.WriteLine("Variável Global: " + variavelGlobal);
}
}

using System;
