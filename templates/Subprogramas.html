Tópico:Subprogramas em C#

Os subprogramas, também conhecidos como rotinas ou funções, são blocos de código independentes que podem ser chamados e executados em um programa. Eles facilitam a modularidade, reutilização de código e a organização de lógica complexa.

Subtópico: Introdução

A introdução aos subprogramas abrange suas características fundamentais. Os subprogramas podem ser funções (retornam um valor) ou procedimentos (não retornam um valor). Eles podem ser chamados várias vezes em diferentes partes do programa.

Subtópico: Características

As características dos subprogramas incluem a capacidade de encapsular lógica, aceitar parâmetros de entrada, executar instruções e, em alguns casos, retornar um valor.

Subtópico: Definições Básicas

A definição básica de um subprograma envolve nome, parâmetros (se houver) e corpo. O corpo contém as instruções a serem executadas.

Subtópico: Parâmetros

Os parâmetros permitem a comunicação entre o chamador e o subprograma. Eles podem ser de entrada, saída ou ambos. Funções podem retornar valores via parâmetros de saída.

Subtópico: Funções e Procedimentos

Funções retornam um valor, enquanto procedimentos não retornam um valor. As funções podem ser usadas em expressões, enquanto os procedimentos são chamados apenas por seu efeito colateral.

Subtópico: Variáveis Locais

Variáveis locais são usadas dentro do subprograma e não são acessíveis fora dele. Elas têm escopo limitado à execução do subprograma.

Subtópico: Subprogramas Aninhados

Subprogramas podem ser aninhados dentro de outros subprogramas. Isso ajuda a organizar a lógica complexa e a melhorar a modularidade.

Subtópico: Passagem de Parâmetros

A passagem de parâmetros permite que informações sejam transmitidas entre o chamador e o subprograma. Isso pode ser feito por valor, referência ou combinações.

Subtópico: Implementação de Passagem de Parâmetro

A implementação de passagem de parâmetros envolve determinar como os valores são transmitidos entre o chamador e o subprograma. Isso afeta a manipulação dos parâmetros.

Subtópico: Verificação de Tipo

A verificação de tipo é importante para garantir que os argumentos passados correspondam aos tipos esperados pelos parâmetros.

Subtópico: Array Multidimensional

Arrays multidimensionais podem ser passados como parâmetros para subprogramas, permitindo a manipulação de matrizes e estruturas de dados complexas.

Subtópico: Subprograma como Parâmetros

Os subprogramas também podem ser passados como parâmetros para outros subprogramas. Isso é útil para criar abstrações de alto nível.

EXEMPLOS DE CODIGO

Exemplo de Código: Definição de Função em C#

class Program {
static int Soma(int a, int b) {
return a + b;
}

static void Main(string[] args) {
int resultado = Soma(3, 5);
Console.WriteLine(resultado); // Resultado: 8
}
}

Exemplo de Código: Definição de Procedimento em C#

class Program {
static void Saudacao(string nome) {
Console.WriteLine("Olá, " + nome + "!");
}

static void Main(string[] args) {
Saudacao("Maria"); // Saída: Olá, Maria!
}
}

Exemplo de Código: Passagem de Parâmetro por Valor em C#

class Program {
static void Dobro(int num) {
num *= 2;
Console.WriteLine("Dentro da função: " + num);
}

static void Main(string[] args) {
int valor = 5;
Dobro(valor);
Console.WriteLine("Fora da função: " + valor); // Saída: Fora da função: 5
}
}

Exemplo de Código: Passagem de Parâmetro por Referência em C#

class Program {
static void Dobro(ref int num) {
num *= 2;
Console.WriteLine("Dentro da função: " + num);
}

static void Main(string[] args) {
int valor = 5;
Dobro(ref valor);
Console.WriteLine("Fora da função: " + valor); // Saída: Fora da função: 10
}
}

Exemplo de Código: Subprograma Aninhado em C#

class Program {
static void Externo() {
Console.WriteLine("Início do subprograma externo");
Interno();
Console.WriteLine("Fim do subprograma externo");
}

static void Interno() {
Console.WriteLine("Subprograma interno");
}

static void Main(string[] args) {
Externo();
}
}

Exemplo de Código: Passagem de Subprograma como Parâmetro em C#

class Program {
static void MostrarMensagem() {
Console.WriteLine("Olá, mundo!");
}

static void AplicarSubprograma(Action subprograma) {
subprograma();
}

static void Main(string[] args) {
AplicarSubprograma(MostrarMensagem); // Saída: Olá, mundo!
}
}
