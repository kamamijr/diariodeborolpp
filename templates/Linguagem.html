Tópico: Linguagem Funcional em C#

A abordagem da linguagem funcional em C# é um paradigma que se concentra na utilização de funções como blocos de construção fundamentais. Este paradigma valoriza a avaliação de expressões e a imutabilidade dos dados, promovendo a criação de código mais legível, modular e seguro. Alguns conceitos-chave da linguagem funcional em C# incluem:

Funções Anônimas:

A capacidade de criar funções sem atribuir um nome a elas, o que é especialmente útil para passá-las como argumentos para outras funções.

Expressões Lambda:

São funções anônimas concisas e de uma única expressão. São amplamente usadas para criar funções de ordem superior e aplicar operações em coleções de dados.

Funções de Ordem Superior:

Essas são funções que podem receber outras funções como argumentos e/ou retorná-las como resultados. Isso possibilita a criação de abstrações poderosas e reutilizáveis.

Imutabilidade:

No paradigma funcional, os dados são tratados como imutáveis, ou seja, não podem ser alterados após serem criados. Isso reduz a complexidade, ajuda a evitar efeitos colaterais indesejados e torna o código mais previsível.

Recursão:

A recursão é uma técnica onde uma função chama a si mesma para resolver problemas de maneira iterativa. É uma parte fundamental da programação funcional e pode ser usada para resolver diversos tipos de problemas.

A programação funcional em C# oferece uma alternativa poderosa ao paradigma imperativo tradicional, permitindo abordagens mais declarativas na criação de código. Ela é especialmente adequada para lidar com problemas complexos que envolvem manipulação de dados, transformações e análises. Ao utilizar os princípios da linguagem funcional em C#, os desenvolvedores podem criar software mais eficiente, modular e resiliente.

EXEMPLOS DE CODIGO

1. Funções Anônimas:

Linguagem
using System;

class Program {
static void Main() {
Func add = (a, b) => a + b;
Console.WriteLine("A soma de 3 e 5 é: " + add(3, 5));
}
}

2. Expressões Lambda:

Linguagem
using System;

class Program {
static void Main() {
Func square = x => x * x;
Console.WriteLine("O quadrado de 4 é: " + square(4));
}
}

3. Funções de Ordem Superior:

Linguagem
using System;
using System.Linq;

class Program {
static void Main() {
int[] numbers = { 1, 2, 3, 4, 5 };
Func isEven = x => x % 2 == 0;

var evenNumbers = numbers.Where(isEven);
Console.WriteLine("Números pares: " + string.Join(", ", evenNumbers));
}
}

4. Imutabilidade:

Linguagem
using System;
using System.Collections.Immutable;

class Program {
static void Main() {
var originalList = ImmutableList.Empty;
var newList = originalList.Add(1).Add(2).Add(3);

Console.WriteLine("Lista original: " + string.Join(", ", originalList));
Console.WriteLine("Nova lista: " + string.Join(", ", newList));
}
}

5. Recursão:

Linguagem
using System;

class Program {
static int Factorial(int n) {
if (n == 0 || n == 1)
return 1;
else
return n * Factorial(n - 1);
}

static void Main() {
int result = Factorial(5);
Console.WriteLine("O fatorial de 5 é: " + result);
}
}
