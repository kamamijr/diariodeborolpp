Tópico:Tipo de Dados em C#

Os tipos de dados são essenciais para definir o tipo de valor que uma variável pode conter. Eles determinam como os dados são armazenados e manipulados na memória.

Subtópico: Introdução

A introdução aos tipos de dados abrange sua importância para a representação e manipulação correta dos valores em um programa. Os tipos garantem que as operações sejam realizadas de acordo com as regras definidas.

Subtópico: Tipos Primitivos

Os tipos primitivos são os blocos de construção fundamentais para a construção de estruturas de dados mais complexas. Eles incluem inteiros, ponto flutuante, booleanos, caracteres, strings, números complexos e decimais.

Subtópico: Tipos Ordinais

Os tipos ordinais representam valores discretos, como enumerações e conjuntos. Eles têm uma ordem específica e são frequentemente usados em decisões de programação.

Subtópico: Array

Arrays são coleções de elementos do mesmo tipo, acessíveis através de um índice. Eles são usados para armazenar dados relacionados.

Subtópico: Tipo Registro

O tipo registro é uma estrutura de dados que pode conter vários campos com diferentes tipos de dados. Ele é usado para agrupar informações relacionadas.

Subtópico: Tipo União

O tipo união permite armazenar diferentes tipos de dados em um único espaço de memória. Existem uniões livres e uniões discriminadas, dependendo da forma como são acessadas.

Subtópico: Tipo Ponteiro

Os tipos ponteiro permitem armazenar o endereço de memória de uma variável. Eles são usados para acessar e manipular dados indiretamente.

Subtópico: Tipo Referência

Os tipos de referência armazenam referências a objetos na memória. Eles são usados para gerenciar objetos dinamicamente alocados.

Subtópico: Verificação de Tipo

A verificação de tipo é o processo de garantir que as operações sejam aplicadas apenas a tipos compatíveis. Isso evita erros de tipo e comportamento inesperado.

exemplo de Código: Tipos Primitivos em C#

using System;

class Program {
static void Main(string[] args) {
int idade = 25;
double salario = 1500.75;
bool ativo = true;
char genero = 'M';
string nome = "Ana";
decimal valor = 120.50M;
Complexo complexo = new Complexo(3, 4);

Console.WriteLine("Idade: " + idade);
Console.WriteLine("Salário: " + salario);
Console.WriteLine("Ativo: " + ativo);
Console.WriteLine("Gênero: " + genero);
Console.WriteLine("Nome: " + nome);
Console.WriteLine("Valor: " + valor);
Console.WriteLine("Complexo: " + complexo.Real + " + " + complexo.Imaginario + "i");
}
}

exemplo de Código: Array em C#

using System; class Program {
static void Main(string[] args) {
int[] numeros = new int[5];
numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;
numeros[3] = 40;
numeros[4] = 50;

for (int i = 0; i < numeros.Length; i++) {
Console.WriteLine("Número " + (i + 1) + ": " + numeros[i]);
}
}
}

exemplo de Código: Tipo Registro em C#

using System;

struct Pessoa {

Console.WriteLine("Idade: " + pessoa1.Idade);

exemplo de Código: Tipo Referência em C#

using System;

class Pessoa {
public string Nome;
public int Idade;
}

class Program {
static void Main(string[] args) {
Pessoa pessoa1 = new Pessoa();
pessoa1.Nome = "Maria";
pessoa1.Idade = 25;

Pessoa pessoa2 = pessoa1; // Cria uma referência, não uma cópia

Console.WriteLine("Pessoa 1 - Nome: " + pessoa1.Nome + ", Idade: " + pessoa1.Idade);
Console.WriteLine("Pessoa 2 - Nome: " + pessoa2.Nome + ", Idade: " + pessoa2.Idade);

pessoa2.Nome = "Ana"; // Altera também o nome da pessoa1

Console.WriteLine("Pessoa 1 - Nome: " + pessoa1.Nome + ", Idade: " + pessoa1.Idade);
Console.WriteLine("Pessoa 2 - Nome: " + pessoa2.Nome + ", Idade: " + pessoa2.Idade);
}
}
