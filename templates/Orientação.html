Tópico: Orientação a Objetos em C#

A Orientação a Objetos (OO) é um paradigma de programação que organiza o código em objetos, que são instâncias de classes. Essa abordagem promove a reutilização de código, modularidade e facilita a modelagem do mundo real no software. Em C#, a OO é amplamente suportada e aplicada.

Subtópico: Introdução

A Orientação a Objetos introduz os conceitos de classes e objetos. Uma classe é um modelo para criar objetos, encapsulando propriedades (atributos) e comportamentos (métodos). Os objetos são instâncias das classes, possuindo estado e comportamento.

Subtópico: Herança

A herança permite criar uma nova classe que herda características (atributos e métodos) de uma classe existente. Isso promove a reutilização de código e a criação de hierarquias de classes. A classe derivada (subclasse) herda características da classe base (superclasse) e pode adicionar suas próprias características.

Subtópico: Conceitos de OO

Os pilares da OO são encapsulamento, abstração, herança e polimorfismo. Encapsulamento protege os detalhes internos da classe. Abstração foca nos aspectos essenciais, ignorando detalhes irrelevantes. Herança permite criar hierarquias de classes. Polimorfismo permite que diferentes classes sejam tratadas de forma uniforme através de interfaces ou classes base.

Subtópico: Exemplo de C++ e Java

Linguagens como C++ e Java também suportam a Orientação a Objetos. Em C++, classes são definidas com métodos e atributos, mas a gerência de memória é mais manual. Java é fortemente orientada a objetos, com gerência de memória automática e ênfase na reutilização de código através de classes e interfaces.

Subtópico: Implementação

Em C#, a implementação da Orientação a Objetos envolve a definição de classes com atributos e métodos, criação de objetos através de construtores e o uso de conceitos como encapsulamento e herança. A OO é fundamental para criar código modular, reutilizável e de fácil manutenção.

Exemplo de Código: Herança em C#

class Animal {
public string Nome { get; set; }
public virtual void EmitirSom() {
Console.WriteLine("Som genérico de animal");
}
}

class Cachorro : Animal {
public override void EmitirSom() {
Console.WriteLine("Latido do cachorro");
}
}

class Gato : Animal {
public override void EmitirSom() {
Console.WriteLine("Miado do gato");
}
}

Exemplo de Código: Abstração em C#

abstract class Forma {
public abstract double CalcularArea();
}

class Circulo : Forma {
public double Raio { get; set; }
public override double CalcularArea() {
return Math.PI * Raio * Raio;
}
}

class Retangulo : Forma {
public double Largura { get; set; }
public double Altura { get; set; }
public override double CalcularArea() {
return Largura * Altura;
}
}

Exemplo de Código: Polimorfismo em C#

class Animal {
public virtual void FazerBarulho() {
Console.WriteLine("Algum som de animal");
}
}

class Cachorro : Animal {
public override void FazerBarulho() {
Console.WriteLine("Latido do cachorro");
}
}

class Gato : Animal {
public override void FazerBarulho() {
Console.WriteLine("Miado do gato");
}
}

Exemplo de Código: Uso de Classes e Objetos em C#

class Pessoa {
public string Nome { get; set; }
public int Idade { get; set; }
}

class Program {
static void Main(string[] args) {
Pessoa pessoa1 = new Pessoa();
pessoa1.Nome = "João";
pessoa1.Idade = 25;

Pessoa pessoa2 = new Pessoa();
pessoa2.Nome = "Maria";
pessoa2.Idade = 30;

Console.WriteLine($"{pessoa1.Nome} tem {pessoa1.Idade} anos.");
Console.WriteLine($"{pessoa2.Nome} tem {pessoa2.Idade} anos.");
} }
