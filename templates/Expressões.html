Tópico: Expressões e Sentenças de Atribuição em C#

As expressões e sentenças de atribuição são fundamentais para manipular dados e realizar cálculos em um programa C#. Expressões combinam valores e operadores para produzir novos valores, enquanto sentenças de atribuição permitem armazenar ou atualizar valores em variáveis. Este tópico aborda expressões aritméticas, condicionais, relacionais, booleanas e a operação de atribuição.

Subtópico: Expressões Aritméticas


As expressões aritméticas envolvem operações matemáticas básicas, como adição, subtração, multiplicação e divisão, permitindo a manipulação de números em um programa.

Subtópico: Expressões Condicionais


As expressões condicionais combinam valores e operadores relacionais para produzir um resultado baseado em uma condição. Elas são frequentemente usadas para tomar decisões em um programa.

Subtópico: Expressões Relacionais


As expressões relacionais comparam valores e determinam a relação entre eles, como igualdade, desigualdade, maior que, menor que, etc.
Subtópico: Expressões Booleanas As expressões booleanas envolvem valores booleanos (verdadeiro ou falso) e operadores lógicos, como AND, OR e NOT, para criar condições mais complexas.

Subtópico: Atribuição


As sentenças de atribuição permitem armazenar valores em variáveis ou atualizar o valor de uma variável existente.

EXEMPLOS DE CODIGO

int a = 5;
int b = 3;
int soma = a + b;
int produto = a * b;
double divisao = (double)a / b; // Conversão para evitar divisão inteira
int idade = 20;
string status = (idade >= 18) ? "Adulto" : "Menor de idade";
int x = 10;
int y = 5;
bool maiorQue = x > y;
bool igual = x == y;
bool condicao1 = true;
bool condicao2 = false;
bool resultado = condicao1 && !condicao2;
int numero = 7;
string nome = "João";
double salario = 1500.50;
numero = 10; // Atualizando o valor da variável "numero"
