Tópico: Estruturas de Controle em C#

As estruturas de controle em C# permitem controlar o fluxo de execução de um programa, permitindo que certas instruções sejam executadas condicionalmente ou repetidamente. Elas são essenciais para a lógica e o controle de um programa, garantindo que as ações corretas sejam tomadas em diferentes situações. As principais estruturas de controle são instruções de seleção e instruções iterativas.

Subtópico: Instrução de Seleção



A instrução de seleção permite executar diferentes trechos de código com base em condições específicas. Em C#, as instruções de seleção incluem a escolha entre dois caminhos usando o if, else if e else. Também há a seleção múltipla usando o switch.

Subtópico: Instrução Iterativa



A instrução iterativa permite a repetição de um bloco de código enquanto uma condição é verdadeira. Em C#, existem duas categorias principais de loops: controlados por contadores (como for) e controlados logicamente (como while e do-while).

Subtópico: Controle Definido pelo Usuário



O controle definido pelo usuário refere-se à capacidade do programa de reagir a entradas do usuário. Isso é feito usando estruturas de seleção para verificar as entradas e executar ações apropriadas com base nesses valores.

Subtópico: Iteração sobre Estrutura



A iteração sobre uma estrutura refere-se a percorrer seus elementos um por um. Isso é frequentemente usado para processar listas, arrays e outras coleções de dados.

EXEMPLOS DE CODIGO

int idade = 18;
if (idade >= 18)
{
Console.WriteLine("Pode votar.");
}
else
{
Console.WriteLine("Não pode votar.");
}




for (int i = 0; i < 5; i++)
{
Console.WriteLine("Valor de i: " + i);
}



int contador = 0;
while (contador < 3)
{
Console.WriteLine("Contador: " + contador);
contador++;
}


Console.Write("Digite um número: ");
int numero = int.Parse(Console.ReadLine());

if (numero % 2 == 0)
{
Console.WriteLine("O número é par.");
}
else
{
Console.WriteLine("O número é ímpar.");
}



int[] numeros = { 1, 2, 3, 4, 5 };
foreach (int num in numeros)
{
Console.WriteLine(num);
}
