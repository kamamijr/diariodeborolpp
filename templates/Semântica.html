Tópico: Semântica em C#

A semântica em programação refere-se ao significado de um programa, tanto em termos de estrutura quanto de comportamento. Ela pode ser dividida em semântica estática e semântica dinâmica. A semântica estática lida com análises feitas antes da execução, como verificações de tipos e escopo. A semântica dinâmica envolve a interpretação do comportamento do programa durante a execução.

Subtópico: Semântica Estática

A semântica estática lida com aspectos do programa que podem ser verificados sem a necessidade de executar o código. Isso inclui análises de tipo, verificação de escopo, resolução de sobrecarga de funções e detecção de erros antes da execução.

Subtópico: Gramáticas de Atributos

As gramáticas de atributos são usadas na semântica estática para associar informações a nós de uma árvore sintática, como anotações de tipos. Elas permitem propagar informações e realizar verificações durante a análise do programa.

Subtópico: Semântica Dinâmica

A semântica dinâmica concentra-se no comportamento do programa durante a execução. Isso inclui a interpretação do significado das instruções, avaliação de expressões e interações entre partes do programa.

Subtópico: Semântica Operacional

A semântica operacional descreve como as instruções são executadas passo a passo em um programa. Ela inclui a especificação de regras de avaliação para cada construção da linguagem, como atribuições, condicionais e loops.

Subtópico: Semântica Denotacional

A semântica denotacional fornece uma abordagem matemática para descrever o comportamento de programas. Ela utiliza funções matemáticas para relacionar construções de linguagem a valores e comportamentos.
Este resumo oferece uma visão abrangente dos diferentes aspectos da semântica em programação, cobrindo semântica estática, gramáticas de atributos, semântica dinâmica, semântica operacional e semântica denotacional. Agora, vou fornecer exemplos de código relacionados a esses tópicos.

EXEMPLOS DE CODIGO

Exemplo de Código: Semântica Estática - Verificação de Tipos em C#

class Program {
static void Main(string[] args) {
int numero = "texto"; // Erro de tipo: atribuição inválida
}
}

Exemplo de Código: Gramáticas de Atributos em C#

class Aluno {
public string Nome { get; set; }
public int Nota { get; set; }
}

class Program {
static void Main(string[] args) {
Aluno aluno = new Aluno();
aluno.Nome = "Maria";

aluno.Nota = "A"; // Erro de tipo: atribuição inválida
}
}

Exemplo de Código: Semântica Dinâmica - Avaliação de Expressões em C#

class Program {
static void Main(string[] args) {
int a = 5;
int b = 3;
int soma = a + b;
Console.WriteLine(soma); // Resultado: 8
}
}

Exemplo de Código: Semântica Operacional - Avaliação de Condicional em C#

class Program {
static void Main(string[] args) {
int idade = 18;
if (idade >= 18) {
Console.WriteLine("Maior de idade");
} else {
Console.WriteLine("Menor de idade");
}
}
}

Exemplo de Código: Semântica Denotacional - Função em C#

class Program {
static int Funcao(int x) {
return 2 * x + 1;
}

static void Main(string[] args) {
int resultado = Funcao(3); // Resultado: 7
Console.WriteLine(resultado);
}
}
